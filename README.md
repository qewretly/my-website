Сайт-визитка

Разработан как итоговая работа по специальности «Информационные системы и программирование».

Технологии
- HTML5: Структура страниц
- CSS3: Стилизация и адаптивный дизайн
- JavaScript: Интерактивные элементы

Описание
- Цветовая палитра: черный фон с зелеными акцентами
- Шрифты: Arial для заголовков, Roboto для текста
- Лэйаут: Горизонтальный с использованием блоков
- Адаптивность: Полноценная поддержка desktop

Структура сайта
1. Главная страница — личная информация и контакты
2. Практика — описание производственной практики
3. Курсовая работа — информация о проекте
4. Сертификаты — достижения и награды
5. Контакты — адрес, телефон, информация о колледже
